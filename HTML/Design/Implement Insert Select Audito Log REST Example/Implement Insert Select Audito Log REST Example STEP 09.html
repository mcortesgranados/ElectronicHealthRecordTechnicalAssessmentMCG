9. Run the API

After implementing all necessary files, follow these steps to install dependencies and run the API.

Install Dependencies

npm install express sequelize mysql2 dotenv

Start the Server

npm run dev

Testing the API

Use Postman or cURL to test API endpoints.

Insert a Log

curl -X POST http://localhost:3000/api/audit-logs \
     -H "Content-Type: application/json" \
     -d '{
           "user_id": "some-user-id",
           "ehr_mapping_id": "some-mapping-id",
           "action": "Update Record",
           "changes": { "field": "old_value -> new_value" }
         }'

Get All Logs

curl -X GET http://localhost:3000/api/audit-logs

Get a Log by ID

curl -X GET http://localhost:3000/api/audit-logs/{id}
