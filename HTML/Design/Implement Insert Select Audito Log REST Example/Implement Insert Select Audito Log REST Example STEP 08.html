Initialize Sequelize in src/infra/db/config/database.ts

This document provides details on setting up Sequelize to connect to a MySQL database.

Sequelize Configuration

import { Sequelize } from 'sequelize';

export const sequelize = new Sequelize(
  process.env.DB_NAME as string,
  process.env.DB_USER as string,
  process.env.DB_PASSWORD as string,
  {
    host: process.env.DB_HOST,
    dialect: 'mysql',
    port: Number(process.env.DB_PORT),
    logging: false,
  }
);

sequelize.sync()
  .then(() => console.log('Database connected'))
  .catch((err) => console.error('Database connection error:', err));

Configuration Details

  • Sequelize Instance: Initializes a new Sequelize instance using environment variables.
  • Dialect: Specifies MySQL as the database dialect.
  • Port: Reads the database port from the environment variables.
  • Logging: Disabled for cleaner output.
  • Synchronization: Ensures that Sequelize synchronizes with the database schema.
