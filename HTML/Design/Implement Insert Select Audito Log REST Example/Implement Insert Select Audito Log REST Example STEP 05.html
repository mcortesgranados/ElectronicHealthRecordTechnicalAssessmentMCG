Audit Log Routes Documentation

This document provides details on the API endpoints for managing audit logs.

Audit Log Routes Implementation

import { Router } from 'express';
import AuditLogController from '../controllers/AuditLogController';

const router = Router();

// Route to create a new audit log entry
router.post('/audit-logs', AuditLogController.createLog);

// Route to fetch all audit logs
router.get('/audit-logs', AuditLogController.getLogs);

// Route to fetch a specific audit log by ID
router.get('/audit-logs/:id', AuditLogController.getLogById);

export default router;

API Endpoints

  • POST /audit-logs - Creates a new audit log entry.
  • GET /audit-logs - Retrieves all audit logs.
  • GET /audit-logs/:id - Retrieves a specific audit log entry by ID.
