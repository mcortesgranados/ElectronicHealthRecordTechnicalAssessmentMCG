7. Environment Configuration (.env)

This document provides details on setting up the environment configuration for the application.

.env File Configuration

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=yourdatabase
DB_PORT=3306

Configuration Details

  • DB_HOST: Specifies the database host, usually localhost for local development.
  • DB_USER: Database username.
  • DB_PASSWORD: Database password.
  • DB_NAME: Name of the database being used.
  • DB_PORT: Port on which the database service is running (default: 3306 for MySQL).
