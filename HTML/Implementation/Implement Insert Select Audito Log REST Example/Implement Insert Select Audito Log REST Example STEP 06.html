6. Register Routes in index.ts

This document provides details on how to register API routes in the main entry file of the application.

index.ts Implementation

import express from 'express';
import auditLogRoutes from './api/routes/auditLogs';

const app = express();

// Middleware to parse JSON requests
app.use(express.json());

// Register audit log routes under /api
app.use('/api', auditLogRoutes);

// Define the server port
const PORT = process.env.PORT || 3000;

// Start the server
app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});

API Setup

  • Express Server: The application initializes an Express server.
  • Middleware: Uses express.json() to parse JSON request bodies.
  • Route Registration: The audit log routes are mounted under /api.
  • Server Listening: The server listens on port 3000 or a custom environment-defined port.
